### 项目基本信息

- **目标形式**：桌面应用（已完成 Electron 打包）
- **技术方向**：Electron 单进程（main.js + preload.js）

### 功能规划
1. **调酒教学模块**：手法教程（Shake/Stir/Build）+ 器具指南，配合 AI 生成写实风格图片
2. **酒单/配方系统**：鸡尾酒配方，每步详细操作，含用量（ml）+ 成本
3. **库存管理**：原料库存录入、库存预警
4. **成本核算**：单杯成本计算、建议售价建议（成本占比 25-30%）
